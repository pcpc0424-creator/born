## 1. 로그인

관리자 페이지에 접속하면 로그인 화면이 나타납니다.

- **아이디**와 **비밀번호**를 입력 후 로그인 버튼을 클릭합니다.
- 로그인 후 **2시간** 동안 활동이 없으면 자동으로 로그아웃됩니다.
- 로그아웃은 우측 상단의 사용자 메뉴에서 할 수 있습니다.

---

## 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 전체 현황을 한눈에 파악할 수 있습니다.

### 통계 카드 (4개)
| 항목 | 설명 |
|------|------|
| 진행 중 행사 | 현재 활성 상태인 행사 수 |
| 전체 회원 | 등록된 총 회원 수 + 오늘 신규 가입 수 |
| 여권 제출 | 여권사본을 제출한 회원 수 |
| 설문 응답 | 설문에 응답한 회원 수 |

### 진행 중인 행사 목록
- 행사명, 일정, 참가 인원, D-Day가 표시됩니다.
- 행사명을 클릭하면 해당 행사 에디터로 이동합니다.

### 최근 가입 회원
- 최근 가입한 회원의 이름, 연락처, 참여 행사가 표시됩니다.

### 빠른 실행 버튼
- **새 행사 등록**, **회원 추가**, **설문 만들기**, **여권 확인** 버튼으로 바로 이동할 수 있습니다.

## 3. 행사 에디터

행사를 생성하고 관리하는 핵심 페이지입니다.

### 화면 구성
- **좌측**: 등록된 행사 목록 (검색 가능)
- **우측**: 선택한 행사의 상세 정보 편집 폼

### 새 행사 등록하기
1. 좌측 상단의 **[+ 새 행사]** 버튼을 클릭합니다.
2. 우측 폼에 행사 정보를 입력합니다.
3. 하단의 **[저장]** 버튼을 클릭합니다.

### 입력 항목

#### 기본 정보
| 항목 | 필수 | 설명 |
|------|:----:|------|
| 행사명 | O | 행사 이름 |
| 시작일 | O | 출발일 (날짜 선택) |
| 종료일 | O | 도착일 (날짜 선택) |
| 상태 | | 활성/비활성 선택 |
| 추가 출발일 | | 여러 출발일이 있는 경우 추가 |

#### 항공 정보
| 항목 | 설명 |
|------|------|
| 항공사 | 이용 항공사명 |
| 출발편 / 귀국편 | 항공편명 (예: KE001) |
| 출발 시간 / 도착 시간 | 각 편의 출발/도착 시각 |
| 출발 공항 / 도착 공항 | 공항명 및 공항 코드 |
| 시차 | 현지와의 시차 (시간 단위) |
| 비행 소요시간 | 출발편/귀국편 각각 (분 단위) |
| 수하물 안내 | 수하물 규정 안내 |
| 항공 기타 안내 | 추가 항공 관련 안내사항 |

#### 미팅 정보
| 항목 | 설명 |
|------|------|
| 미팅 장소 | 공항 미팅 위치 |
| 미팅 시간 | 미팅 시각 |
| 미팅 날짜 | 미팅 날짜 |
| 인솔자 | 담당 인솔자 이름 |
| 인솔자 연락처 | 담당 인솔자 전화번호 |
| 미팅 안내사항 | 추가 미팅 관련 안내 |

#### 콘텐츠
| 항목 | 설명 |
|------|------|
| 거래처 로고 | 거래처 로고 이미지 업로드 (로그인 화면에 표시) |
| 여행 유의사항 | 여행 전 안내사항 (클릭하면 넓은 입력창 표시) |
| 출국 준비물 | 체크리스트 형태로 표시됨 (줄바꿈으로 항목 구분) |
| 기내 반입 금지 물품 | 줄바꿈으로 항목 구분 |
| 날씨 이미지 | 현지 날씨 이미지 업로드 |

### 행사 관리 기능
- **QR 코드**: 행사 등록 링크의 QR 코드를 생성하고 다운로드할 수 있습니다.
- **링크 복사**: 행사 고유 링크를 클립보드에 복사합니다. 이 링크를 참가자에게 공유하면 됩니다.
- **삭제**: 행사를 삭제합니다. (관련 데이터도 함께 삭제됨)

## 4. 행사-개인 에디터

행사별 참가자를 관리하고, 버스/석식/객실 배정을 하는 페이지입니다.

### 사용 방법
1. 상단 드롭다운에서 **행사를 선택**합니다.
2. 해당 행사의 참가자 목록이 표시됩니다.

### 참가자 추가
- **[회원 추가]** 버튼 클릭 → 이름/전화번호로 검색 → 선택하여 추가
- **[엑셀 업로드]** 버튼으로 일괄 등록 가능

### 참가자 목록 표시 항목
| 항목 | 설명 |
|------|------|
| 이름 | 한글명 + 영문명 |
| 생년월일 | YYYY.MM.DD 형식 |
| 선택관광 | 신청한 선택관광이 뱃지로 표시 |
| 버스 (일차별) | 행사 일수만큼 컬럼이 생성됨. 직접 입력 가능 |
| 전화번호 | 포맷된 전화번호 |
| 석식 테이블 | 테이블 번호 직접 입력 |
| 객실 번호 | 호텔 객실 번호 직접 입력 |
| 삭제 | 참가자 제거 |

### 실시간 자동 저장
- 버스 번호, 석식 테이블, 객실 번호는 **입력 즉시 자동 저장**됩니다.
- 별도의 저장 버튼을 누를 필요가 없습니다.

### 엑셀 내보내기
- **[엑셀 다운로드]** 버튼으로 참가자 목록을 엑셀 파일로 저장할 수 있습니다.

## 5. 선택관광 에디터

행사별 선택관광 프로그램을 등록하고 관리합니다.

### 사용 방법
1. 상단 드롭다운에서 **행사를 선택**합니다.
2. **[선택관광 추가]** 버튼을 클릭합니다.

### 입력 항목
| 항목 | 필수 | 설명 |
|------|:----:|------|
| 선택관광명 | O | 프로그램 이름 |
| 금액 | | 참가 비용 (0원이면 "무료"로 표시) |
| 소요시간 | | 예: "약 2시간" |
| 미팅시간 | | 선택관광 집합 시간 |
| 진행 일차 | | 행사 일수에 맞춰 체크박스 표시. 복수 선택 가능 |
| 설명 | | 상세 설명 (최대 1000자) |
| 유의사항 | | 참가 시 주의사항 (최대 1000자) |
| 상태 | | 활성/비활성 |

### 참가자 측 동작
- 사용자 페이지에서 참가자가 직접 원하는 선택관광을 체크하여 신청합니다.
- 신청 현황은 **행사-개인 에디터**에서 확인할 수 있습니다.

---

## 6. 일정표 에디터

행사의 일차별 상세 일정을 작성합니다.

### 사용 방법
1. 좌측 행사 목록에서 행사를 선택합니다.
2. 행사 기간에 맞춰 **일차별 카드**가 자동 생성됩니다.
3. 각 일차에 일정 항목을 추가합니다.

### 일정 항목 입력
- 시간, 장소, 활동 내용, 참고사항 등을 입력할 수 있습니다.
- 일정 항목의 순서를 변경할 수 있습니다.

행사별 숙박 호텔 정보를 등록합니다.

### 사용 방법
1. 상단 드롭다운에서 **행사를 선택**합니다.
2. **[호텔 추가]** 버튼을 클릭합니다.

### 입력 항목
| 항목 | 필수 | 설명 |
|------|:----:|------|
| 호텔명 | O | 호텔 이름 |
| 주소 | | 호텔 주소 |
| 전화번호 | | 호텔 연락처 |
| 체크인 날짜/시간 | | 체크인 정보 |
| 체크아웃 날짜/시간 | | 체크아웃 정보 |
| 설명 | | 호텔 안내사항 (최대 1000자) |

| 정렬 순서 | | 표시 순서 (숫자가 작을수록 위에 표시) |

## 8. 공지/문의 에디터

참가자에게 보여줄 공지사항과 FAQ를 관리합니다.

### 탭 구성
- **공지사항 탭**: 행사 관련 안내사항
- **FAQ 탭**: 자주 묻는 질문과 답변

### 사용 방법
1. 원하는 탭을 선택합니다.
2. **[추가]** 버튼을 클릭합니다.
3. 제목과 내용을 입력하고 저장합니다.

### 입력 항목
| 항목 | 필수 | 설명 |
|------|:----:|------|
| 제목 | O | 공지/FAQ 제목 |
| 내용 | | 상세 내용 (클릭하면 넓은 입력창 표시, 최대 1000자) |

### 정렬 순서
- 목록에서 **정렬 순서** 숫자를 직접 수정하면 표시 순서가 변경됩니다.
- 숫자가 작을수록 위에 표시됩니다.

## 9. 설문 에디터

참가자 대상 설문을 만들고 관리합니다.

### 설문 만들기
1. 상단 드롭다운에서 **[+ 새 설문]** 을 선택합니다.
2. 설문 제목을 입력하고 저장합니다.

### 설문 구조
설문은 **페이지** > **질문** 구조로 되어 있습니다.

- 여러 페이지로 나눠서 질문을 구성할 수 있습니다.
- **[페이지 추가]** 버튼으로 새 페이지를 만듭니다.

### 질문 유형
| 유형 | 설명 |
|------|------|
| 객관식 | 선택지 중 하나를 고르는 문항 (선택지를 쉼표로 구분하여 입력) |
| 단답형 | 짧은 텍스트 입력 |
| 서술형 | 긴 텍스트 입력 |

### 질문 추가
1. 원하는 페이지에서 **[질문 추가]** 를 클릭합니다.
2. 질문 내용, 유형, 선택지(객관식인 경우)를 입력합니다.
3. 필수 응답 여부를 설정합니다.

---

## 10. 설문 통계

설문 응답 결과를 확인합니다.

### 사용 방법
1. 상단 드롭다운에서 설문을 선택합니다.
2. 응답 통계가 자동으로 표시됩니다.

### 통계 표시 방식
| 질문 유형 | 표시 방식 |
|-----------|-----------|
| 객관식 | 도넛 차트 + 선택지별 응답 수/비율 |
| 단답형/서술형 | 응답자별 텍스트 목록 |

### 요약 정보
- 총 응답자 수
- 총 질문 수

---

## 11. 회원 관리

전체 회원을 조회, 등록, 수정, 삭제합니다.

### 회원 검색
- 상단 검색창에 **이름, 전화번호, 아이디**로 검색할 수 있습니다.
- 행사별 필터로 특정 행사 참가자만 조회할 수 있습니다.

### 회원 추가
1. **[회원 추가]** 버튼을 클릭합니다.
2. 회원 정보를 입력합니다.

### 입력 항목
| 항목 | 필수 | 설명 |
|------|:----:|------|
| 아이디 | O | 로그인용 아이디 (중복 불가) |
| 비밀번호 | O | 로그인 비밀번호 |
| 한글 이름 | O | 한글명 |
| 영문 이름 | | 여권상 영문명 |
| 성별 | | 남/여 |
| 생년월일 | | 날짜 선택 |
| 전화번호 | | 연락처 |

### 엑셀 기능
- **[엑셀 다운로드]**: 현재 검색/필터된 회원 목록을 엑셀 파일로 저장
- **[엑셀 업로드]**: 엑셀 파일로 회원 일괄 등록 (이름+생년월일로 중복 확인)

## 12. 여권사본 확인

행사별 참가자의 여권사본 제출 현황을 확인합니다.

### 사용 방법
1. 상단 드롭다운에서 **행사를 선택**합니다.
2. 참가자별 여권 제출 현황이 표시됩니다.

### 통계 카드
| 항목 | 설명 |
|------|------|
| 전체 인원 | 행사 참가자 수 |
| 제출 완료 | 여권사본을 제출한 인원 (녹색) |
| 미제출 | 여권사본 미제출 인원 (적색) |
| 제출률 | 제출 비율 (%) |

### 여권 정보 표시
- 이름, 전화번호, 여권번호(일부 마스킹), 생년월일, 만료일, 제출일이 표시됩니다.
- **민감 정보는 암호화**되어 저장되며, 화면에서는 일부만 표시됩니다.

### 기능
- **여권 이미지 보기**: 제출된 여권사본 이미지를 확인할 수 있습니다.
- **이미지 일괄 다운로드**: 모든 여권사본을 ZIP 파일로 다운로드합니다.
- **엑셀 다운로드**: 여권 정보 목록을 엑셀로 저장합니다.

## 13. 페이지 노출 관리

행사별로 사용자 페이지에서 어떤 메뉴를 보여줄지 설정합니다.

### 사용 방법
1. 상단 드롭다운에서 **행사를 선택**합니다.
2. 각 메뉴의 **토글 스위치**를 켜거나 끕니다.
3. 변경 즉시 자동 저장됩니다.

### 설정 가능 항목
| 메뉴 | 설명 |
|------|------|

## 참가자 등록 흐름 (전체 프로세스)

아래는 행사를 만들고 참가자가 이용하기까지의 전체 흐름입니다.

```
1. 행사 에디터에서 새 행사 등록
       ↓
2. 행사 정보 입력 (항공, 미팅, 유의사항 등)
       ↓
3. 회원 관리에서 참가자 등록 (또는 엑셀 업로드)
       ↓
4. 행사-개인 에디터에서 참가자를 행사에 배정
       ↓
5. 선택관광, 일정표, 호텔 정보 등록
       ↓
6. 페이지 노출 관리에서 보여줄 메뉴 설정
       ↓
7. 행사 에디터에서 QR코드/링크 복사 → 참가자에게 공유
       ↓
8. 참가자가 링크로 접속 → 회원가입/로그인 → 행사 정보 확인
```

## 자주 하는 작업

### 행사 링크를 참가자에게 공유하려면?
1. 행사 에디터 → 해당 행사 선택
2. 우측 상단 **[링크 복사]** 또는 **[QR 코드]** 클릭
3. 복사된 링크 또는 QR 이미지를 카카오톡 등으로 전달

### 참가자를 한꺼번에 등록하려면?
1. 회원 관리 → **[엑셀 업로드]** 클릭
2. 양식에 맞춰 엑셀 파일 작성 후 업로드
3. 행사-개인 에디터에서 해당 행사에 배정

### 특정 메뉴를 참가자에게 숨기려면?
1. 페이지 노출 관리 → 행사 선택
2. 숨기고 싶은 메뉴의 토글을 끄기

### 설문 응답을 확인하려면?
1. 설문 통계 → 해당 설문 선택
2. 차트와 텍스트 응답 확인
